# FastPortScanner
Python3 fast and stealth port scanner for pentesting.
Stealther alternative for nmap SYN scan.

## Requirements:

The only requirement in this tool is Colorama, you can install it using:<br>
`~$ pip3 install colorama`
## Usage:

Linux and MacOS:<br>
`~$ python3 FastPortScanner.py`

Windows:<br>
`python FastPortScanner.py`
